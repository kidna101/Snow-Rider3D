# Bike Rider 3D

A free browser-based 3D downhill bike riding game. Steer left and right, dodge obstacles, and collect items to unlock new bikes and worlds.

**[▶ Play now](https://kidna101.github.io/Snow-Rider3D/)**

---

## Controls

| Key | Action |
|-----|--------|
| ← Arrow / A | Steer left |
| → Arrow / D | Steer right |

## Worlds & Progression

| World | Unlock |
|-------|--------|
| Mountain / Alpine | Available from the start |
| Illawarra Coastal Forest | Ride 2 km cumulative total |

| Bike | Unlock |
|------|--------|
| Mountain Bike | Available from the start |
| Cruiser Bike | Collect 20 items total |

Progress is saved in your browser (localStorage). It won't carry over between different browsers or devices.

---

## Tech

- [Three.js](https://threejs.org/) 0.163 — 3D rendering in the browser
- Vanilla JavaScript (ES modules) — no build step required
- GitHub Pages — free hosting

## Local development

```bash
npx serve .
# Open http://localhost:3000
```
